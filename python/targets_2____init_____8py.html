Detailed Description

init.py file tells python that this folder is package.

all list defined in init.py tells which modules names should be imported form package import * Exports following modules :- 1) install 2) interface 3) enforce 4) parse 5) rewrite 6) slice 7) run 8) show 9) strip 10) previrt 11) watch = 12) specialize 13) help 14) peval = 15) args = Specializes the arguments by partial evaluation. 16) pack 17) build 18) clang 19) as = 20) ar = Archiving libraries that llvm can link to make linking fast. 21) cp 22) ld

Another variable is also defined in the init.py which is map . Basically it is dictionary that maps the tool name given in the command line to the tool that is available in the occam toolchain For example:- clang++ -> clang clang -> clang clang-cpp -> clang All the above commands are converted to one tool.
