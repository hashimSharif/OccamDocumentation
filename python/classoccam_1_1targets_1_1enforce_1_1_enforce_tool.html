Detailed Description

This class executes opt(llvm) with Penforce pass that Enforcing an interface can be particularly useful for externally defined functions (system and library calls).

Restricting the usages of the functions helps enhance security.

Executes opt(llvm) tool with -Penforce pass that enforce all of the given interfaces on input.bc .

This pass takes one of two options :-

1) Penforce-inside = Enforce an interface inside a function. Code is injected into function definitions.

2) Penforce-outside = Enforce an interface outside a call.(function declaration)
