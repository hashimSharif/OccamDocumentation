Executes occam's BuildTool using llvm-ld tool.

It opens .manifest which have necessary configurations, arguments flags and libraries path given in it. This information makes linking of bitcode file possible. This routine extracts following informaiton from .manifest file:-

1) Native libraries paths

2) Linker Flags

3) Configuration settings of paticular application.
