  output_file,
  opts 
)

Runs opt(llvm) with all Dead Code elimination passes.These are standard passes.

opt passes used in this function are defined below:- -globaldce = Dead code elimination. This transform is designed to eliminate unreachable internal globals from the program. -strip = Strip all symbols from a module.Strip utility is used for reducing code size. -globalopt = Global Variable Optimizer.This pass transforms simple global variables that never have their address taken. -strip-dead-prototypes = Strip Unused Function Prototypes.This pass loops over all of the functions in the input module, looking for dead declarations and removes them
