Detailed Description

    Wrapper for llvm archiving tool to make linking fast. @b llvm-ar is to common UNIX utility ar.
    It archive several files together into single file. The intent for this is to produce archive libraries LLVM 
    bitcodde that can be linked into an LLVM program. @b llvm-ar generates a symbol table that makes 
    linking faster because only the symbol table needs to be consulted, not each individual file 
    member of the archive.
