Executes BUILD Tool which excutes llvm-ld on the bitcode file.

The basic steps of execution are as follows:- 1) First find list of path given in command line argument 2) Then list of libraries from argumnts 3) Then list of native libraries 4) Then runs llvm ld tool Method is used by clang and ld tool defined in occam toolchain.
