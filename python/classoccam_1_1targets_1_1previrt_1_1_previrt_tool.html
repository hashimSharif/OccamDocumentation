Detailed Description

Pre-virtualize a compilation unit based on a manifest file sent as argument to the PrevirtTool Procedure of Pre-virtualization is as follows:

1) Read the configuration in the manifest file.

2) Create watch functions if they are requested.

3) Invoke Passes.interface() that executes the 'Pinterface2' pass with opt which compute the external dependencies of the module, implemented in the C++ library Class previrt::GatherInterface2Pass

4) Invoke passes.intern() : Internalizes the interfaces and perform Dead Code Elimination invoking the 'Poccam' Pass implemented in the C++ library Class previrt::OccamPass.

5) Invoke passes.peval() : Apply a Partial Evaluation Pass, creating specialized functions based on specific usages by invoking the 'Ppeval' Pass, implemented in the C++ library Class ParEvalOptPass .

6) Invoke passes.interface to compute all the external references made from within the modules.

9) Invoke passes.specialize() to create specialized versions of functions based on the Gathered Interfaces that in turn invokes the 'Pspecialize' Pass implemented in the C++ library Class SpecializeComponentPass

10) Invoke passes.rewrite()to rewrite all call-sites corresponding to the specialized functions by invoking the 'Prewrite' Pass implemented in the C++ library Class previrt::RewriteComponentPass

11) prune Dead code.

12) llvm-ld = LLVM linking tool.
