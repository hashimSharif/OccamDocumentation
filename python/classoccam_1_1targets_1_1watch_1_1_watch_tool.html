Detailed Description

This tool watch the given module with the given watch specifications.

For debugging purpose, The Pass creates watch functions for functions that are a part of the watch::proto::WatchInterface. It enforces a certain interface as specified in the watch::proto::WatchInterface Google protocol buffer.

The Pass takes four command line arguments.

watch::proto::WatchInterface serialized files are passed following the 'Pwatch2-input' flag. The 'Pwatch2-local' flag if passed, specifies that local calls to the function should be allowed. The 'Pwatch2-fail' flag is followed by the name of the failure function, that is called in case the interface is violated. If the 'Pwatch2-fail' flag is specified, the failure function is created is a fancy failure function; it takes as argument, the name of the function that violated the interface and the set of arguments.
