Runs LLVM ar tool to archive the inputs.

llvm-ar is to common UNIX utility ar. It archive several files together into single file. The intent for this is to produce archive libraries LLVM bitcode that can be linked into an LLVM program.

def occam.toolchain.findlib (   l,
  paths 
)

Search the l(library) in paths argument.

First find in OCCAM_LIB_PATH environmental variable, then make path of library and check if this path exists. then call occam.toolchain.getSearchPath() to find path again.

Returns null if library is not found in paths list.
