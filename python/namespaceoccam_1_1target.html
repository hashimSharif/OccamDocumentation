def occam.target.register (   name,
  tool 
)

Each Derived Tool that inherits the Target Class, announces it's availability by calling the register method.

The register method takes the name of the Tool and maps the object of the Tool against the name. Once occam is invoked with the tool Name, the TOOLS dictionary is looked up with the name of the Tool and the corresponding objects run method is invoked.

def occam.target.run (   args,
  tool = None 
)

Top level wrapper Function to run the tool present in occam toolchain.

The basic steps performed in this target.run() are as follows:-

1) Takes tool name from CLI ;Finds respective tool occam.targets directory and imports in that tool module. 2) Extract tool from TOOLS dictionary then call run method on it. 3) If tool is not present in TOOLS dict and OCCAM_protect_path variable is set; it will execute occam.driver.runUnknown().
