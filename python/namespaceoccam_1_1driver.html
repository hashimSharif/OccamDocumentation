def occam.driver.runUnknown (   tool,
  args 
)

When any tool was run and tool module is not found in targets package, then this function is called.

Finds the tool path given in argument and executes using subprocess. Finds the path of tool using OCCAM_PROTECTED_PATH environmental variable and then open subprocess using that tool path. OCCAM_PROTECT_PATH environmental variable is when set, occam will remove all non-occam items from the path and intercept calls to common executables so that they can be logged at level INFO. Can be useful for debugging a build that isn't working properly.

Parameters
tool=tool name that user want to run
args=names of arguments to tool that wanted to be run.
Returns
ReturnCode of subprocess which shows status of program. 
    Returns Path after if argument path exits.
