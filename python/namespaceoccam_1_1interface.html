def occam.interface.joinInterfaces (   into,

The function merges the "merge" ComponentInterface into the "into" ComponentInterface, The ComponentInterface buffer logs the exteral function calls and the references.

The joinInterfaces routine applies merging and returns a single ComponentInterface object that contains all the external function calls and the references that exist in both the ComponentInterface buffers.
