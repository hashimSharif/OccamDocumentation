Detailed Description

The sliceTool calls the 'Poccam' Pass implemented in the class previrt::OccamPass to internalize all functions and performs Dead Code Elimination.

Removes all the functionality that is not necessary to implement the given interface. interface is an object of type previrt::proto::ComponentInterface, containing an array of previrt::proto::CallInfo and a set of references (defining function names). Function names are required by the Poccam pass to correctly eliminate all unused code across the modules.
