Detailed Description

Install modules and packages in default /use/local/lib/ directory so that it can be made accesscible from anywhere.

This file call distutils.core.setup() to install following packages:-

  1. occam
  2. occam.targets
  3. occam.proto

setup.py is a python file, which usually tells you that the module/package you are about to install have been packaged and distributed with Distutils, which is the standard for distributing Python Modules. This allows easily install packages. python setup.py install
