Detailed Description

A CallRewrite class contains the rewrite information, to be applied to function calls. CallRewrite objects are used in the class previrt::RewriteComponentPass. To apply a rewrite the name of the specialized function and the reduced set of arguments is required, that are a part of a CallRewrite object
