ConstSpecScheme  l,

static

A map of specialization *.

Returns true if 'r' is at least as strict a specialization policy as 'l'. The ConstSpecScheme array, contains NULL values against arguments that are not be specialized and the values of the arguments to the specialized. 'r' is stricter than 'l', if 'r' has at-least as many NULL values as 'l', and whenever and index in 'r' is NON-NULL (a value for an argument) ,it matches the corresponding value in 'l' ConstSpecScheme.
