Detailed Description

RecursiveGuardSpecPolicy implements a policy to guard against the specialization of recursive functions. The constructor takes as an argument an AggressiveSpecPolicy or ConservativeSpecPolicy object (to specialize the arguments to the functions), and the call graph of the Module. It iterates the call graph identifying all recursive functions, and stores all found recursive into a class member map sccs. Recursive functions are important to identify, as they cannot be specialized. Specializing recursive functions could lead to a very large number of specialized versions.

The specializeOn function is a virtual function, with separate implementations in the derived classes ConservativeSpecPolicy, AggressiveSpecPolicy and RecursiveGuardSpecPolicy. The detailed descriptions of each implementation are as follows:

ConservativeSpecPolicy::specializeOn , AggressiveSpecPolicy::specializeOn detailed description:

The specializeOn method, returns a Value **. Value ** is essentially a array defining the specialization policy against each argument. If an argument is a constant and is can be partially evaluated into the function body, the argument is returned in the array. For arguments cannot be partially evaluated into the function body, their corresponding index in the Value* array is NULL.

RecursiveGuardSpecPolicy::specializeOn detailed description:

The function guards the specialization of recursive functions. It Returns a policy array consisting of a policy against every argument to the function. The array contains NULL values against arguments that cannot be specialized and NON-NULL values against arguments that are to be specialized. The function in turn calls the specializeOn method of the delegate class member. Delegate can be an object of two possible classes: ConservativeSpecPolicy and AggressiveSpecPolicy. If the function is a recursive function, NULL is returned, as recursive functions cannot be specialized. (specialization could result in a large number of specialized versions of the recursive function)
