Detailed Description

The RegisterArguments class is a derived class over SpecializeArguments.It is essentially a wrapper. SpecializeArguments constructor requires a set of arguments: filename,progName RegisterArguments constructor extracts the filename and the progName from the command line lists and calls the SpecializeArguments constructor.

RegisterArguments is registered as a pass with the PassManager internal database by the name 'Parguments'. Two arguments are required by the pass. A file name followed by the flag 'Parguments-input' specifies the input file containing the program arguments.

The program Name is also input to the pass and is followed by the 'Parguments-name' flag.
