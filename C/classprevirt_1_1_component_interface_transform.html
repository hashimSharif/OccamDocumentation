Detailed Description

The ComponentInterfaceTransform class essentially contains two objects. The interface object (data class member) of type previrt::ComponentInterface, contains the set of external calls and references. The external calls and references are gathered by the 'Pinterface2' Pass, implemented in the class previrt::GatherInterface2Pass. The class member rewrites, is a map of function names against CallInfo and CallRewrite, that is required by the 'Prewrite' Pass implemented in the class previrt::RewriteComponentPass.
