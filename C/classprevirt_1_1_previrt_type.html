Detailed Description

The PrevirtType class is a wrapper above the previrt::proto::PrevirtType buffer. The format of the previrt::proto::PrevirtType is defined in the file Previrt.proto line 78. The class can contain values of the type int, string, float, vector or a global value. The PrevirtType class extends the functionality of the previrt::proto::PrevirtType class, by adding a number of utility functions.

Build an equality function to compare two values. The Type of the buffer, determines whether a string or int equality function is required. The cacheEq type keeps a map of Function * against Type *.
