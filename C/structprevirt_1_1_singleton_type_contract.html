Detailed Description

The SingletonTypeContract defines the arguments and methods to add entry tests to functions. The class data member params is a vector of previrt::PrevirtType arguments and the method SingletonTypeContract::emitEntryTest adds an entry test to a function,(testing equality of each previrt::PrevirtType argument against the corresponding function parameters)
