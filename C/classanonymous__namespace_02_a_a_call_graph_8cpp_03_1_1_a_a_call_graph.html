Detailed Description

The AACallGraph class creates a static call graph of the module. The previrt::GatherInterface2 Class uses CallGraph owing to the benefit that we can (in theory) hook into alias analysis and other passes that refine indirect function calls. Alias Analysis (aka Pointer Analysis) is a class of techniques which attempt to determine whether or not two pointers ever can point to the same object in memory
