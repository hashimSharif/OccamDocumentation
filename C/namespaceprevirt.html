bool previrt::MinimizeComponent ( Module &  M,
ComponentInterface &  I 
)

Remove all code from the given module that is not necessary to implement the given interface. In llvm each function is by default external-linkage(assumed to be called externally). To remove dead code it is necessary to modify the linkage of the functions, not called externally to internal-linkage. previrt::localizeLinkage is called to return an internal-linkage counter-part for each external definition. Once internalized the llvm PassManager performs dead code elimination in the module. The dead code elimination is performed in a loop, until there is more code still to be removed.

The given Interface I of type previrt::proto::ComponentInterface is a list of functions and Globals that are accessed from other modules. These functions and globals should not be internalized as they are being accessed from other modules.
