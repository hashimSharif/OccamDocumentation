Detailed Description

The StripGccComment class removes the Gcc comment from the inline asm, so that no information of the version of Gcc that compiled the code is exposed.

virtual bool previrt::StripGccComment::runOnModule ( Module &  m)
inlinevirtual

The routine extracts the inline Asm inside the module as a string. If no Gcc comment(.ident) was found inside the string, false is returned. If a comment was found, the line containing the comment, is removed and the modified inline Asm string excluding the Gcc comment is inserted into the module.
