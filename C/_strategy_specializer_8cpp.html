static bool trySpecializeFunction ( Function *  f,
SpecializationTable &  table,

std::list< Function * > &  to_add 
)
static

The trySpecializeFunction iterates over all of the instructions in the function, looking for call instructions. If the call contains constant arguments, a specialized copy of the function corresponding to the arguments is looked up in the previrt::SpecializationTable (storing a list of applied specializations). If the table does not contain a version corresponding to the particular usage, a new cloned copy of the function is created, with arguments directly evaluated in the body of the function, hence fewer parameters to the function. The specialized version is added to the previrt::SpecializationTable for future reference.
