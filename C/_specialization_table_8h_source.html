65  /*! The SpecializationTable is a repository of specializations. If a specialized version of a function has been created, specific
66  to a set of arguments, there is no need to create another version, if the function is called with exactly the same arguments.
67  Therefore before a specialized copy of a function is required, it is looked up in the SpecializationTable. If a function corresponding
68  to the specialization exists, it is retrieved from the table, otherwise previrt::specializeFunction is called to create
69  a new specialized version with respect to the usage.
70  */
